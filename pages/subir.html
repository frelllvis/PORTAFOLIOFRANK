ℹ️ ¿Cómo funciona el almacenamiento?
Los archivos se guardan en el almacenamiento local del navegador (localStorage) de este dispositivo. Para que el portafolio sea visible en GitHub Pages con tus trabajos, deberás volver a subir los archivos cada vez que uses un nuevo dispositivo. Los datos de perfil y curso también se guardan localmente.
